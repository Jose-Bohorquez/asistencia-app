=================================================================
                        INSTRUCCIONES DE USO
=================================================================

1. ACCESO AL SISTEMA:
   - URL: http://localhost:81/asistencia-app/public/
   - Usar las credenciales de arriba según el rol que desees probar

2. FUNCIONALIDADES POR ROL:

   SUPER ADMIN:
   - Dashboard completo con estadísticas generales
   - Módulo de Usuarios: CRUD completo
   - Módulo de Programas: CRUD completo
   - Módulo de Cursos: CRUD completo
   - Módulo de Sesiones: CRUD completo
   - Reportes avanzados y exportaciones
   - Configuración del sistema

   ADMIN:
   - Dashboard con estadísticas limitadas
   - Gestión de usuarios (sin eliminar)
   - Gestión completa de programas y cursos
   - Gestión de sesiones y asistencias
   - Reportes y exportaciones

   PROFESOR:
   - Dashboard personal con sus cursos
   - Ver solo sus cursos asignados
   - Crear y gestionar sesiones de sus cursos
   - Tomar asistencia de sus estudiantes
   - Reportes de sus cursos únicamente

3. PRUEBAS RECOMENDADAS:

   a) Probar login con cada rol
   b) Verificar que cada rol solo vea sus opciones permitidas
   c) Intentar acceder a funciones no permitidas (debe redirigir)
   d) Probar creación, edición y eliminación según permisos
   e) Verificar exportaciones y reportes
   f) Probar funcionalidad de logout

4. NOTAS IMPORTANTES:
   - Las contraseñas están hasheadas en la base de datos
   - El sistema tiene middleware de seguridad activo
   - Todas las acciones se registran en logs
   - El sistema valida permisos en cada acción

=================================================================
                         SOLUCIÓN DE PROBLEMAS
=================================================================

Si tienes problemas para acceder:

1. Verificar que la base de datos esté corriendo
2. Verificar que los usuarios estén insertados en la tabla 'usuarios'
3. Verificar que el servidor web esté funcionando
4. Revisar logs en caso de errores

Para resetear contraseñas:
- Las contraseñas están hasheadas con password_hash() de PHP
- Para crear nuevas: usar password_hash('nueva_contraseña', PASSWORD_DEFAULT)
